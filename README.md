ROAST.exe is an AI-powered roast machine engineered to deliver brutal, personalized roasts. Input your bio, photo, GitHub, or LinkedIn profile, choose your persona and intensity, and prepare for digital destruction.

**Live Demo:** [https://roast-exe.vercel.app/](https://roast-exe.vercel.app/)
**GitHub:** [https://github.com/instai4/ROAST.EXE](https://github.com/instai4/ROAST.EXE)

---

### ✨ Features

*   **4 Input Modes:**
    *   Bio/Description text
    *   Photo upload (drag & drop enabled)
    *   GitHub URL
    *   LinkedIn URL
*   **3 Roast Intensities:**
    *   Mild (friendly jab)
    *   Savage (no filter)
    *   Nuclear (absolute chaos)
*   **8 Roaster Personas:**
    *   Comedian
    *   Gordon Ramsay
    *   Shakespeare
    *   Drill Sergeant
    *   Therapist
    *   Gen-Z
    *   Cold AI
    *   Villain
*   **Gemini Vision API** integration for photo-based roasting.
*   **Animated Burn Score Meter** (1 to 10).
*   **Copy Roast to Clipboard** with one click.
*   **Direct Share to X (Twitter)** functionality.
*   **Roast Again** button for fresh roasts with the same input.
*   **Screen Shake Animation** triggered by Nuclear intensity.
*   **Multi-Provider AI Fallback:** Grok then Groq then Gemini, ensuring robust roast delivery.

### 🚀 How to Use

1.  Choose your input mode: Bio, Photo, GitHub URL, or LinkedIn URL.
2.  Paste your information or upload a photo.
3.  Select your desired roast intensity.
4.  Pick a roaster persona.
5.  Hit the **ROAST ME** button.
6.  Read your destruction.
7.  Share it directly to X if you're brave enough.

### ⚙️ Installation

To set up ROAST.exe locally or deploy it:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/instai4/ROAST.EXE.git
    cd ROAST.EXE
    
2.  **Deploy to Vercel:** This project is optimized for Vercel Serverless deployment.
3.  **Add Environment Variables:** Configure the following API keys in your Vercel project settings or local `.env` file:
    *   `XAI_API_KEY`
    *   `GROQ_API_KEY`
    *   `GEMINI_API_KEY`
4.  **API Endpoint:** Ensure the `api/roast.js` file is correctly placed in your `/api` directory for Vercel Serverless functions.
5.  **Access:** Open the live URL provided by Vercel and prepare to suffer.
```
